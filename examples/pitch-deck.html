SlideSmith

Forge editable PowerPoint from HTML

Local-first · No limits · Free & open source

The Problem

Current HTML-to-PPTX tools are either cloud-only, expensive, or produce non-editable screenshots.

Screenshot approach

Text becomes images. Can't edit, search, or resize without quality loss.

SaaS tools

Upload your data to the cloud. Pay monthly. Hit slide limits.

Manual rebuild

Recreate everything in PowerPoint from scratch. Time-consuming and error-prone.

How It Works

Three steps, all local

1

Write HTML

Use CSS flexbox, grid, gradients — anything the browser supports.

2

Run SlideSmith

One command. Browser renders, DOM is traversed, PPTX is generated.

3

Edit in PowerPoint

Text is text. Shapes are shapes. Fully editable, no screenshots.
